# Superstore Sales Analysis using Power BI

This project analyzes Superstore sales data using Microsoft Power BI to understand sales trends and business insights.

## Tools Used
- Microsoft Power BI
- CSV Dataset
- Data Visualization

## Dashboard Preview
![Dashboard](https://github.com/ap-darshan/PowerBI-Superstore-Sales-Dashboard/blob/main/Power%20BI%20Dashboard.png)

## Key Insights
- Sales by Category
- Sales by Segment
- Sales by Region
- Sales by Sub-Category
- Sales Distribution by State
